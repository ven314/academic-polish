# 学术润色 — Academic Polish

中英文学术论文润色与降重助手。自动识别输入语言，提供三版本对照输出。

## 功能

- **中文学术润色**：句式重组、同义替换、语态调整、冗余删减
- **英文学术润色**：sentence restructure, vocabulary upgrade, grammar fix
- **多版本对照**：保守版（最小改动） / 中度版（中等改写） / 激进版（大幅降重）
- **语言自动识别**：输入中文按中文规则处理，英文按英文规则处理

## 使用方式

### 标准润色

```
> 帮我润色这段话：我们采用了基于深度学习的方法来进行桥梁损伤检测。实验结果表明了该方法的可行性。

## 润色结果
采用基于深度学习的桥梁损伤检测方法。实验结果表明，该方法具有良好的可行性。
```

### 多版本对照

```
> 帮我润色这段话，多版本对照：我用了Z24桥梁的数据来验证我的方法。数据有1530个样本，分为17种损伤状态。

## 多版本对照
| 版本 | 原文 | 润色后 |
|------|------|--------|
| 保守版 | ... | ... |
| 中度版 | ... | ... |
| 激进版 | ... | ... |
```

### 英文润色

```
> Can you polish this paragraph: This paper proposes a method for bridge damage detection...

## Polished Text
This paper presents a method for bridge damage detection...
```

## 触发关键词

| 语言 | 触发词 |
|------|--------|
| 中文 | 润色、降重、改表达、改学术、改得专业点、优化表达、口语化、让它更学术 |
| 英文 | polish, rewrite, improve the English/writing, proofread, sound more academic |

## 不触发场景

写新内容、翻译、格式化参考文献、语法检查、列目录结构等不触发此 skill。

## 三版本规则

| 版本 | 改写力度 | 目标查重降幅 |
|------|---------|------------|
| 保守版 | 仅改语法、口语词 | 5-10% |
| 中度版 | 句式重组 + 同义替换 | 15-25% |
| 激进版 | 全面重写 | 30-50% |
